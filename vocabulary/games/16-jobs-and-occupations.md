## 🎮 遊戲設計

### 遊戲：職業大轉盤

**遊戲類型：** 🎰 SPIN 轉盤抽獎 — 轉盤停在單字上回答問題

**遊戲說明：**

畫面中央有一個彩色轉盤，上面寫著不同的英文單字。小朋友按下按鈕讓轉盤旋轉，停下來後需要正確唸出或選出該單字的意思。答對可獲得星星獎勵！

**遊戲流程：**

1. 畫面中央顯示彩色轉盤，分成多個扇形區域
2. 每個扇形區域寫著一個英文單字
3. 小朋友按下 SPIN 按鈕或滑動讓轉盤旋轉
4. 轉盤逐漸減速並停在一個單字上
5. 系統詢問該單字的意思，顯示 2~3 個選項
6. 答對：獲得該單字對應的獎勵道具
7. 答錯：顯示正確答案，轉盤再轉一次機會

**難度分級：**

| 難度 | 參數設定 |
|------|----------|
| ⭐ Easy | 轉盤 4 格，答題選項 2 個，可重複作答 |
| ⭐⭐ Medium | 轉盤 6 格，答題選項 3 個，錯誤顯示正確答案 |
| ⭐⭐⭐ Hard | 轉盤 8 格，答題選項 4 個，限時 5 秒作答 |

**Flame 技術實作要點：**

- 使用 `RotateEffect` 實現轉盤旋轉和減速停止動畫
- 使用 `TimerComponent` 控制轉盤減速曲線和停止時機
- 使用 `SpriteComponent` 繪製轉盤扇形區域和指針
- 使用 `TextComponent` 在扇形區域顯示單字
- 使用 `TapCallbacks` 處理 SPIN 按鈕和答題選項的點擊
- 使用 `ScaleEffect` 實現停止時單字的強調動畫

**素材需求：**

- 轉盤素材：彩色轉盤底圖（可分 4/6/8 格版本）
- 指針素材：轉盤指針圖片
- 圖片素材：30 組單字對應圖片
- 音效素材：轉盤旋轉音效、停止音效、答對/答錯音效
- 動畫素材：轉盤旋轉粒子效果、獎品彈出動畫
- UI 素材：SPIN 按鈕、答題選項面板

### 遊戲：職業翻翻卡

**遊戲類型：** 🃏 MATCH 記憶翻牌 — 翻牌配對英文與圖片

**遊戲說明：**

畫面上排列著翻面的卡片，卡片背面有可愛圖案。小朋友每次翻開兩張卡片，一張是英文單字、一張是對應圖片，找到配對就得分。訓練記憶力和單字認知。

**遊戲流程：**

1. 畫面顯示 N×M 排列的翻面卡片
2. 小朋友點擊翻開第一張卡片（英文單字或圖片）
3. 點擊翻開第二張卡片嘗試配對
4. 配對成功：兩張卡片發光消失 + 得分
5. 配對失敗：兩張卡片翻回背面
6. 記住卡片位置，用最少次數完成全部配對
7. 完成後根據翻牌次數給予星星評價（1-3 星）

**難度分級：**

| 難度 | 參數設定 |
|------|----------|
| ⭐ Easy | 2x2 共 4 張卡片（2 組配對），翻開後停留 3 秒 |
| ⭐⭐ Medium | 3x2 共 6 張卡片（3 組配對），翻開後停留 2 秒 |
| ⭐⭐⭐ Hard | 4x3 共 12 張卡片（6 組配對），翻開後停留 1 秒 |

**Flame 技術實作要點：**

- 使用 `SpriteComponent` 製作卡片正面（圖片/文字）和背面
- 使用 `FlipEffect`（自訂 `ScaleEffect` X 軸翻轉）模擬翻牌效果
- 使用 `TimerComponent` 控制卡片翻開後的停留時間
- 使用 `ColorEffect` 標記已配對成功的卡片
- 使用 `ShuffleEffect` 在遊戲開始時打亂卡片位置
- 透過 `TapCallbacks` 處理翻牌點擊事件

**素材需求：**

- 卡片正面素材：30 張英文單字卡片 + 30 張對應圖片卡片
- 卡片背面素材：統一的可愛卡背設計（PNG, 200x280px）
- 音效素材：翻牌音效、配對成功音效、配對失敗音效
- 動畫素材：翻牌 3D 翻轉效果、配對成功消除特效
- UI 素材：翻牌次數計數器、星星評價圖示

## 📊 學習目標驗證

### 驗證方式

| 驗證方法 | 說明 | 通過標準 |
|----------|------|----------|
| 聽音辨字 | 播放英文發音，選出正確圖片 | 正確率 ≥ 80% |
| 看圖選字 | 顯示圖片，選出正確英文單字 | 正確率 ≥ 80% |
| 拼寫驗證 | 拖動字母拼出完整單字 | 正確率 ≥ 70% |
| 配對驗證 | 限時內完成英文-圖片配對 | 完成率 100%，翻牌次數 ≤ 最佳次數 ×1.5 |

### 學習進度追蹤

每個單字有獨立的熟練度指標：

| 等級 | 名稱 | 條件 |
|------|------|------|
| ⬜ | 未學習 | 尚未接觸此單字 |
| 🟨 | 認識中 | 在遊戲中正確辨認 1-2 次 |
| 🟩 | 熟悉 | 在不同遊戲中正確辨認 3-5 次 |
| ⭐ | 精熟 | 連續 3 次不同時段皆正確，且反應時間 ≤ 3 秒 |

### 複習機制

- **間隔重複**：根據艾賓浩斯遺忘曲線，在 1 天、3 天、7 天、14 天後自動安排複習
- **弱點加強**：錯誤率高的單字會在後續遊戲中提高出現頻率
- **每週小測驗**：每週自動生成一份包含本週所學單字的小測驗
