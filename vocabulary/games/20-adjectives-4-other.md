## 🎮 遊戲設計

### 遊戲：形容詞泡泡戰

**遊戲類型：** 💥 POP 泡泡消除 — 戳破包含正確單字的泡泡

**遊戲說明：**

畫面中飄浮著許多彩色泡泡，每個泡泡上寫著一個英文單字。播放語音後，小朋友要快速點擊（戳破）包含正確單字的泡泡。泡泡會慢慢飄走，要在消失前戳破！

**遊戲流程：**

1. 播放目標單字的語音和顯示圖片
2. 多個彩色泡泡從畫面四周飄入
3. 每個泡泡上顯示不同的英文單字
4. 小朋友快速點擊正確的泡泡將其戳破
5. 戳破正確泡泡：爆炸星星特效 + 得分
6. 戳破錯誤泡泡：泡泡變灰 + 扣分提示
7. 限時內戳破越多正確泡泡，分數越高

**難度分級：**

| 難度 | 參數設定 |
|------|----------|
| ⭐ Easy | 泡泡移動慢，體積大，同時 3 個泡泡，目標泡泡顏色不同 |
| ⭐⭐ Medium | 泡泡移動速度中等，體積縮小，同時 5 個泡泡 |
| ⭐⭐⭐ Hard | 泡泡移動快，體積小，同時 7 個泡泡，泡泡會重疊 |

**Flame 技術實作要點：**

- 使用 `CircleComponent` 繪製泡泡形狀
- 使用 `RemoveEffect` + `ScaleEffect` 實現戳破動畫
- 使用 `MoveEffect` 讓泡泡隨機飄動
- 使用 `TextComponent` 在泡泡上顯示英文單字
- 使用 `OpacityEffect` 讓泡泡逐漸消失
- 透過 `TapCallbacks` 偵測點擊泡泡事件

**素材需求：**

- 泡泡素材：多種顏色泡泡圖片（PNG, 帶透明度）
- 文字素材：19 個單字的文字渲染
- 音效素材：泡泡破裂音效（pop sound）、背景泡泡音
- 粒子特效：泡泡爆破星星粒子、碎片動畫
- 圖片素材：19 組目標提示圖片

### 遊戲：形容詞轉轉樂

**遊戲類型：** 🎰 SPIN 轉盤抽獎 — 轉盤停在單字上回答問題

**遊戲說明：**

畫面中央有一個彩色轉盤，上面寫著不同的英文單字。小朋友按下按鈕讓轉盤旋轉，停下來後需要正確唸出或選出該單字的意思。答對可獲得星星獎勵！

**遊戲流程：**

1. 畫面中央顯示彩色轉盤，分成多個扇形區域
2. 每個扇形區域寫著一個英文單字
3. 小朋友按下 SPIN 按鈕或滑動讓轉盤旋轉
4. 轉盤逐漸減速並停在一個單字上
5. 系統詢問該單字的意思，顯示 2~3 個選項
6. 答對：獲得該單字對應的獎勵道具
7. 答錯：顯示正確答案，轉盤再轉一次機會

**難度分級：**

| 難度 | 參數設定 |
|------|----------|
| ⭐ Easy | 轉盤 4 格，答題選項 2 個，可重複作答 |
| ⭐⭐ Medium | 轉盤 6 格，答題選項 3 個，錯誤顯示正確答案 |
| ⭐⭐⭐ Hard | 轉盤 8 格，答題選項 4 個，限時 5 秒作答 |

**Flame 技術實作要點：**

- 使用 `RotateEffect` 實現轉盤旋轉和減速停止動畫
- 使用 `TimerComponent` 控制轉盤減速曲線和停止時機
- 使用 `SpriteComponent` 繪製轉盤扇形區域和指針
- 使用 `TextComponent` 在扇形區域顯示單字
- 使用 `TapCallbacks` 處理 SPIN 按鈕和答題選項的點擊
- 使用 `ScaleEffect` 實現停止時單字的強調動畫

**素材需求：**

- 轉盤素材：彩色轉盤底圖（可分 4/6/8 格版本）
- 指針素材：轉盤指針圖片
- 圖片素材：19 組單字對應圖片
- 音效素材：轉盤旋轉音效、停止音效、答對/答錯音效
- 動畫素材：轉盤旋轉粒子效果、獎品彈出動畫
- UI 素材：SPIN 按鈕、答題選項面板

## 📊 學習目標驗證

### 驗證方式

| 驗證方法 | 說明 | 通過標準 |
|----------|------|----------|
| 聽音辨字 | 播放英文發音，選出正確圖片 | 正確率 ≥ 80% |
| 看圖選字 | 顯示圖片，選出正確英文單字 | 正確率 ≥ 80% |
| 拼寫驗證 | 拖動字母拼出完整單字 | 正確率 ≥ 70% |
| 配對驗證 | 限時內完成英文-圖片配對 | 完成率 100%，翻牌次數 ≤ 最佳次數 ×1.5 |

### 學習進度追蹤

每個單字有獨立的熟練度指標：

| 等級 | 名稱 | 條件 |
|------|------|------|
| ⬜ | 未學習 | 尚未接觸此單字 |
| 🟨 | 認識中 | 在遊戲中正確辨認 1-2 次 |
| 🟩 | 熟悉 | 在不同遊戲中正確辨認 3-5 次 |
| ⭐ | 精熟 | 連續 3 次不同時段皆正確，且反應時間 ≤ 3 秒 |

### 複習機制

- **間隔重複**：根據艾賓浩斯遺忘曲線，在 1 天、3 天、7 天、14 天後自動安排複習
- **弱點加強**：錯誤率高的單字會在後續遊戲中提高出現頻率
- **每週小測驗**：每週自動生成一份包含本週所學單字的小測驗
