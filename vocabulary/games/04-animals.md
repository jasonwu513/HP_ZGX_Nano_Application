## 🎮 遊戲設計

### 遊戲：動物記憶王

**遊戲類型：** 🃏 MATCH 記憶翻牌 — 翻牌配對英文與圖片

**遊戲說明：**

畫面上排列著翻面的卡片，卡片背面有可愛圖案。小朋友每次翻開兩張卡片，一張是英文單字、一張是對應圖片，找到配對就得分。訓練記憶力和單字認知。

**遊戲流程：**

1. 畫面顯示 N×M 排列的翻面卡片
2. 小朋友點擊翻開第一張卡片（英文單字或圖片）
3. 點擊翻開第二張卡片嘗試配對
4. 配對成功：兩張卡片發光消失 + 得分
5. 配對失敗：兩張卡片翻回背面
6. 記住卡片位置，用最少次數完成全部配對
7. 完成後根據翻牌次數給予星星評價（1-3 星）

**難度分級：**

| 難度 | 參數設定 |
|------|----------|
| ⭐ Easy | 2x2 共 4 張卡片（2 組配對），翻開後停留 3 秒 |
| ⭐⭐ Medium | 3x2 共 6 張卡片（3 組配對），翻開後停留 2 秒 |
| ⭐⭐⭐ Hard | 4x3 共 12 張卡片（6 組配對），翻開後停留 1 秒 |

**Flame 技術實作要點：**

- 使用 `SpriteComponent` 製作卡片正面（圖片/文字）和背面
- 使用 `FlipEffect`（自訂 `ScaleEffect` X 軸翻轉）模擬翻牌效果
- 使用 `TimerComponent` 控制卡片翻開後的停留時間
- 使用 `ColorEffect` 標記已配對成功的卡片
- 使用 `ShuffleEffect` 在遊戲開始時打亂卡片位置
- 透過 `TapCallbacks` 處理翻牌點擊事件

**素材需求：**

- 卡片正面素材：35 張英文單字卡片 + 35 張對應圖片卡片
- 卡片背面素材：統一的可愛卡背設計（PNG, 200x280px）
- 音效素材：翻牌音效、配對成功音效、配對失敗音效
- 動畫素材：翻牌 3D 翻轉效果、配對成功消除特效
- UI 素材：翻牌次數計數器、星星評價圖示

### 遊戲：動物泡泡樂

**遊戲類型：** 💥 POP 泡泡消除 — 戳破包含正確單字的泡泡

**遊戲說明：**

畫面中飄浮著許多彩色泡泡，每個泡泡上寫著一個英文單字。播放語音後，小朋友要快速點擊（戳破）包含正確單字的泡泡。泡泡會慢慢飄走，要在消失前戳破！

**遊戲流程：**

1. 播放目標單字的語音和顯示圖片
2. 多個彩色泡泡從畫面四周飄入
3. 每個泡泡上顯示不同的英文單字
4. 小朋友快速點擊正確的泡泡將其戳破
5. 戳破正確泡泡：爆炸星星特效 + 得分
6. 戳破錯誤泡泡：泡泡變灰 + 扣分提示
7. 限時內戳破越多正確泡泡，分數越高

**難度分級：**

| 難度 | 參數設定 |
|------|----------|
| ⭐ Easy | 泡泡移動慢，體積大，同時 3 個泡泡，目標泡泡顏色不同 |
| ⭐⭐ Medium | 泡泡移動速度中等，體積縮小，同時 5 個泡泡 |
| ⭐⭐⭐ Hard | 泡泡移動快，體積小，同時 7 個泡泡，泡泡會重疊 |

**Flame 技術實作要點：**

- 使用 `CircleComponent` 繪製泡泡形狀
- 使用 `RemoveEffect` + `ScaleEffect` 實現戳破動畫
- 使用 `MoveEffect` 讓泡泡隨機飄動
- 使用 `TextComponent` 在泡泡上顯示英文單字
- 使用 `OpacityEffect` 讓泡泡逐漸消失
- 透過 `TapCallbacks` 偵測點擊泡泡事件

**素材需求：**

- 泡泡素材：多種顏色泡泡圖片（PNG, 帶透明度）
- 文字素材：35 個單字的文字渲染
- 音效素材：泡泡破裂音效（pop sound）、背景泡泡音
- 粒子特效：泡泡爆破星星粒子、碎片動畫
- 圖片素材：35 組目標提示圖片

## 📊 學習目標驗證

### 驗證方式

| 驗證方法 | 說明 | 通過標準 |
|----------|------|----------|
| 聽音辨字 | 播放英文發音，選出正確圖片 | 正確率 ≥ 80% |
| 看圖選字 | 顯示圖片，選出正確英文單字 | 正確率 ≥ 80% |
| 拼寫驗證 | 拖動字母拼出完整單字 | 正確率 ≥ 70% |
| 配對驗證 | 限時內完成英文-圖片配對 | 完成率 100%，翻牌次數 ≤ 最佳次數 ×1.5 |

### 學習進度追蹤

每個單字有獨立的熟練度指標：

| 等級 | 名稱 | 條件 |
|------|------|------|
| ⬜ | 未學習 | 尚未接觸此單字 |
| 🟨 | 認識中 | 在遊戲中正確辨認 1-2 次 |
| 🟩 | 熟悉 | 在不同遊戲中正確辨認 3-5 次 |
| ⭐ | 精熟 | 連續 3 次不同時段皆正確，且反應時間 ≤ 3 秒 |

### 複習機制

- **間隔重複**：根據艾賓浩斯遺忘曲線，在 1 天、3 天、7 天、14 天後自動安排複習
- **弱點加強**：錯誤率高的單字會在後續遊戲中提高出現頻率
- **每週小測驗**：每週自動生成一份包含本週所學單字的小測驗
