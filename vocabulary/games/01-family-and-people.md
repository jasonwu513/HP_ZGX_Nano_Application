## 🎮 遊戲設計

### 遊戲：家族拼圖

**遊戲類型：** 🧩 DRAG 拖拉配對 — 拖動單字到對應圖片

**遊戲說明：**

畫面左側顯示英文單字卡，右側顯示對應圖片。小朋友用手指拖動單字卡到正確的圖片上進行配對。配對成功時會播放歡呼音效和星星動畫。

**遊戲流程：**

1. 畫面左側顯示 3~5 個英文單字卡片
2. 畫面右側顯示對應數量的圖片
3. 小朋友長按單字卡片開始拖動
4. 拖動到正確圖片上方時，圖片邊框發光提示
5. 放開手指完成配對，播放成功音效
6. 配對錯誤時卡片會彈回原位
7. 全部配對完成後播放過關動畫

**難度分級：**

| 難度 | 參數設定 |
|------|----------|
| ⭐ Easy | 每次配對 2 組，圖片與單字差異明顯，拖動吸附範圍大 |
| ⭐⭐ Medium | 每次配對 3-4 組，加入相似圖片，吸附範圍縮小 |
| ⭐⭐⭐ Hard | 每次配對 5 組，圖片相似度高，需精確拖放，限時 60 秒 |

**Flame 技術實作要點：**

- 使用 `DragCallbacks` mixin 實現拖動功能
- 使用 `HasCollisionDetection` 偵測拖放碰撞區域
- 透過 `RectangleHitbox` 定義卡片和目標的碰撞範圍
- 使用 `MoveEffect` 實現卡片歸位/彈回動畫
- 使用 `GlowEffect` 提示正確放置區域
- 使用 `SequenceEffect` 串連配對成功的動畫序列

**素材需求：**

- 圖片素材：30 組單字對應圖片（PNG, 256x256px）
- 卡片素材：30 張英文單字卡片（含可愛邊框設計）
- 音效素材：拖動音效、放置成功音效、彈回音效
- 動畫素材：配對成功星星特效、發光邊框效果
- UI 素材：進度條、完成面板

### 遊戲：家人翻翻樂

**遊戲類型：** 🃏 MATCH 記憶翻牌 — 翻牌配對英文與圖片

**遊戲說明：**

畫面上排列著翻面的卡片，卡片背面有可愛圖案。小朋友每次翻開兩張卡片，一張是英文單字、一張是對應圖片，找到配對就得分。訓練記憶力和單字認知。

**遊戲流程：**

1. 畫面顯示 N×M 排列的翻面卡片
2. 小朋友點擊翻開第一張卡片（英文單字或圖片）
3. 點擊翻開第二張卡片嘗試配對
4. 配對成功：兩張卡片發光消失 + 得分
5. 配對失敗：兩張卡片翻回背面
6. 記住卡片位置，用最少次數完成全部配對
7. 完成後根據翻牌次數給予星星評價（1-3 星）

**難度分級：**

| 難度 | 參數設定 |
|------|----------|
| ⭐ Easy | 2x2 共 4 張卡片（2 組配對），翻開後停留 3 秒 |
| ⭐⭐ Medium | 3x2 共 6 張卡片（3 組配對），翻開後停留 2 秒 |
| ⭐⭐⭐ Hard | 4x3 共 12 張卡片（6 組配對），翻開後停留 1 秒 |

**Flame 技術實作要點：**

- 使用 `SpriteComponent` 製作卡片正面（圖片/文字）和背面
- 使用 `FlipEffect`（自訂 `ScaleEffect` X 軸翻轉）模擬翻牌效果
- 使用 `TimerComponent` 控制卡片翻開後的停留時間
- 使用 `ColorEffect` 標記已配對成功的卡片
- 使用 `ShuffleEffect` 在遊戲開始時打亂卡片位置
- 透過 `TapCallbacks` 處理翻牌點擊事件

**素材需求：**

- 卡片正面素材：30 張英文單字卡片 + 30 張對應圖片卡片
- 卡片背面素材：統一的可愛卡背設計（PNG, 200x280px）
- 音效素材：翻牌音效、配對成功音效、配對失敗音效
- 動畫素材：翻牌 3D 翻轉效果、配對成功消除特效
- UI 素材：翻牌次數計數器、星星評價圖示

## 🏆 獎勵設計

- ⭐ **星星**：每答對一題獲得 1-3 顆星星（依難度）
- 🎖️ **連擊獎勵**：連續答對 3 題以上，獲得額外金幣

- 收集品可在「我的收藏」頁面查看

## 📊 學習目標驗證

### 驗證方式

| 驗證方法 | 說明 | 通過標準 |
|----------|------|----------|
| 聽音辨字 | 播放英文發音，選出正確圖片 | 正確率 ≥ 80% |
| 看圖選字 | 顯示圖片，選出正確英文單字 | 正確率 ≥ 80% |
| 拼寫驗證 | 拖動字母拼出完整單字 | 正確率 ≥ 70% |
| 配對驗證 | 限時內完成英文-圖片配對 | 完成率 100%，翻牌次數 ≤ 最佳次數 ×1.5 |

### 學習進度追蹤

每個單字有獨立的熟練度指標：

| 等級 | 名稱 | 條件 |
|------|------|------|
| ⬜ | 未學習 | 尚未接觸此單字 |
| 🟨 | 認識中 | 在遊戲中正確辨認 1-2 次 |
| 🟩 | 熟悉 | 在不同遊戲中正確辨認 3-5 次 |
| ⭐ | 精熟 | 連續 3 次不同時段皆正確，且反應時間 ≤ 3 秒 |

### 複習機制

- **間隔重複**：根據艾賓浩斯遺忘曲線，在 1 天、3 天、7 天、14 天後自動安排複習
- **弱點加強**：錯誤率高的單字會在後續遊戲中提高出現頻率
- **每週小測驗**：每週自動生成一份包含本週所學單字的小測驗
