## 🎮 遊戲設計

### 遊戲：日用品接接樂

**遊戲類型：** 🎪 CATCH 接住落下 — 用籃子接住正確單字

**遊戲說明：**

各種物品從畫面上方落下，每個物品上標示英文單字。小朋友左右移動底部的籃子，接住老師指定的正確單字物品，避開錯誤的物品。速度會逐漸加快！

**遊戲流程：**

1. 畫面上方開始掉落帶有英文單字的物品
2. 畫面頂部顯示目標：要接住的單字（圖片+中文提示）
3. 小朋友左右滑動底部籃子移動
4. 接住正確物品：籃子發光 + 加分
5. 接住錯誤物品：籃子震動 + 扣一顆心
6. 漏接正確物品不扣分但不加分
7. 每接住 5 個正確單字，速度略微提升

**難度分級：**

| 難度 | 參數設定 |
|------|----------|
| ⭐ Easy | 物品下落速度慢，同時 1 個物品，籃子寬度大 |
| ⭐⭐ Medium | 下落速度中等，同時 2 個物品，籃子寬度中等 |
| ⭐⭐⭐ Hard | 下落速度快，同時 3 個物品，籃子寬度小，物品會左右飄動 |

**Flame 技術實作要點：**

- 使用 `MoveEffect` 控制物品從上方落下的動畫
- 使用 `CollisionCallbacks` + `RectangleHitbox` 偵測籃子接住物品
- 使用 `DragCallbacks` 或觸控事件控制籃子左右移動
- 使用 `SpawnComponent` 定時從隨機位置生成下落物品
- 使用 `SpriteComponent` 載入物品和籃子的圖片素材
- 使用 `RemoveEffect` 移除已接住或掉落的物品

**素材需求：**

- 物品素材：72 組帶有單字的下落物品圖片
- 籃子素材：可愛籃子/容器圖片（配合主題設計）
- 音效素材：接住音效、漏接音效、背景音樂
- 動畫素材：接住成功閃光效果、物品落地碎裂效果
- UI 素材：分數面板、愛心生命值、速度指示器

### 遊戲：名詞翻翻樂

**遊戲類型：** 🃏 MATCH 記憶翻牌 — 翻牌配對英文與圖片

**遊戲說明：**

畫面上排列著翻面的卡片，卡片背面有可愛圖案。小朋友每次翻開兩張卡片，一張是英文單字、一張是對應圖片，找到配對就得分。訓練記憶力和單字認知。

**遊戲流程：**

1. 畫面顯示 N×M 排列的翻面卡片
2. 小朋友點擊翻開第一張卡片（英文單字或圖片）
3. 點擊翻開第二張卡片嘗試配對
4. 配對成功：兩張卡片發光消失 + 得分
5. 配對失敗：兩張卡片翻回背面
6. 記住卡片位置，用最少次數完成全部配對
7. 完成後根據翻牌次數給予星星評價（1-3 星）

**難度分級：**

| 難度 | 參數設定 |
|------|----------|
| ⭐ Easy | 2x2 共 4 張卡片（2 組配對），翻開後停留 3 秒 |
| ⭐⭐ Medium | 3x2 共 6 張卡片（3 組配對），翻開後停留 2 秒 |
| ⭐⭐⭐ Hard | 4x3 共 12 張卡片（6 組配對），翻開後停留 1 秒 |

**Flame 技術實作要點：**

- 使用 `SpriteComponent` 製作卡片正面（圖片/文字）和背面
- 使用 `FlipEffect`（自訂 `ScaleEffect` X 軸翻轉）模擬翻牌效果
- 使用 `TimerComponent` 控制卡片翻開後的停留時間
- 使用 `ColorEffect` 標記已配對成功的卡片
- 使用 `ShuffleEffect` 在遊戲開始時打亂卡片位置
- 透過 `TapCallbacks` 處理翻牌點擊事件

**素材需求：**

- 卡片正面素材：72 張英文單字卡片 + 72 張對應圖片卡片
- 卡片背面素材：統一的可愛卡背設計（PNG, 200x280px）
- 音效素材：翻牌音效、配對成功音效、配對失敗音效
- 動畫素材：翻牌 3D 翻轉效果、配對成功消除特效
- UI 素材：翻牌次數計數器、星星評價圖示

## 📊 學習目標驗證

### 驗證方式

| 驗證方法 | 說明 | 通過標準 |
|----------|------|----------|
| 聽音辨字 | 播放英文發音，選出正確圖片 | 正確率 ≥ 80% |
| 看圖選字 | 顯示圖片，選出正確英文單字 | 正確率 ≥ 80% |
| 拼寫驗證 | 拖動字母拼出完整單字 | 正確率 ≥ 70% |
| 配對驗證 | 限時內完成英文-圖片配對 | 完成率 100%，翻牌次數 ≤ 最佳次數 ×1.5 |

### 學習進度追蹤

每個單字有獨立的熟練度指標：

| 等級 | 名稱 | 條件 |
|------|------|------|
| ⬜ | 未學習 | 尚未接觸此單字 |
| 🟨 | 認識中 | 在遊戲中正確辨認 1-2 次 |
| 🟩 | 熟悉 | 在不同遊戲中正確辨認 3-5 次 |
| ⭐ | 精熟 | 連續 3 次不同時段皆正確，且反應時間 ≤ 3 秒 |

### 複習機制

- **間隔重複**：根據艾賓浩斯遺忘曲線，在 1 天、3 天、7 天、14 天後自動安排複習
- **弱點加強**：錯誤率高的單字會在後續遊戲中提高出現頻率
- **每週小測驗**：每週自動生成一份包含本週所學單字的小測驗
