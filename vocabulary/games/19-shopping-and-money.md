## 🎮 遊戲設計

### 遊戲：購物清單配對

**遊戲類型：** 🧩 DRAG 拖拉配對 — 拖動單字到對應圖片

**遊戲說明：**

畫面左側顯示英文單字卡，右側顯示對應圖片。小朋友用手指拖動單字卡到正確的圖片上進行配對。配對成功時會播放歡呼音效和星星動畫。

**遊戲流程：**

1. 畫面左側顯示 3~5 個英文單字卡片
2. 畫面右側顯示對應數量的圖片
3. 小朋友長按單字卡片開始拖動
4. 拖動到正確圖片上方時，圖片邊框發光提示
5. 放開手指完成配對，播放成功音效
6. 配對錯誤時卡片會彈回原位
7. 全部配對完成後播放過關動畫

**難度分級：**

| 難度 | 參數設定 |
|------|----------|
| ⭐ Easy | 每次配對 2 組，圖片與單字差異明顯，拖動吸附範圍大 |
| ⭐⭐ Medium | 每次配對 3-4 組，加入相似圖片，吸附範圍縮小 |
| ⭐⭐⭐ Hard | 每次配對 5 組，圖片相似度高，需精確拖放，限時 60 秒 |

**Flame 技術實作要點：**

- 使用 `DragCallbacks` mixin 實現拖動功能
- 使用 `HasCollisionDetection` 偵測拖放碰撞區域
- 透過 `RectangleHitbox` 定義卡片和目標的碰撞範圍
- 使用 `MoveEffect` 實現卡片歸位/彈回動畫
- 使用 `GlowEffect` 提示正確放置區域
- 使用 `SequenceEffect` 串連配對成功的動畫序列

**素材需求：**

- 圖片素材：25 組單字對應圖片（PNG, 256x256px）
- 卡片素材：25 張英文單字卡片（含可愛邊框設計）
- 音效素材：拖動音效、放置成功音效、彈回音效
- 動畫素材：配對成功星星特效、發光邊框效果
- UI 素材：進度條、完成面板

### 遊戲：購物輪盤

**遊戲類型：** 🎰 SPIN 轉盤抽獎 — 轉盤停在單字上回答問題

**遊戲說明：**

畫面中央有一個彩色轉盤，上面寫著不同的英文單字。小朋友按下按鈕讓轉盤旋轉，停下來後需要正確唸出或選出該單字的意思。答對可獲得星星獎勵！

**遊戲流程：**

1. 畫面中央顯示彩色轉盤，分成多個扇形區域
2. 每個扇形區域寫著一個英文單字
3. 小朋友按下 SPIN 按鈕或滑動讓轉盤旋轉
4. 轉盤逐漸減速並停在一個單字上
5. 系統詢問該單字的意思，顯示 2~3 個選項
6. 答對：獲得該單字對應的獎勵道具
7. 答錯：顯示正確答案，轉盤再轉一次機會

**難度分級：**

| 難度 | 參數設定 |
|------|----------|
| ⭐ Easy | 轉盤 4 格，答題選項 2 個，可重複作答 |
| ⭐⭐ Medium | 轉盤 6 格，答題選項 3 個，錯誤顯示正確答案 |
| ⭐⭐⭐ Hard | 轉盤 8 格，答題選項 4 個，限時 5 秒作答 |

**Flame 技術實作要點：**

- 使用 `RotateEffect` 實現轉盤旋轉和減速停止動畫
- 使用 `TimerComponent` 控制轉盤減速曲線和停止時機
- 使用 `SpriteComponent` 繪製轉盤扇形區域和指針
- 使用 `TextComponent` 在扇形區域顯示單字
- 使用 `TapCallbacks` 處理 SPIN 按鈕和答題選項的點擊
- 使用 `ScaleEffect` 實現停止時單字的強調動畫

**素材需求：**

- 轉盤素材：彩色轉盤底圖（可分 4/6/8 格版本）
- 指針素材：轉盤指針圖片
- 圖片素材：25 組單字對應圖片
- 音效素材：轉盤旋轉音效、停止音效、答對/答錯音效
- 動畫素材：轉盤旋轉粒子效果、獎品彈出動畫
- UI 素材：SPIN 按鈕、答題選項面板

## 🏆 獎勵設計

### 即時獎勵
- ⭐ **星星**：每答對一題獲得 1-3 顆星星（依難度）

- 💎 **完美獎勵**：單局全部答對，獲得鑽石 ×1

## 📊 學習目標驗證

### 驗證方式

| 驗證方法 | 說明 | 通過標準 |
|----------|------|----------|
| 聽音辨字 | 播放英文發音，選出正確圖片 | 正確率 ≥ 80% |
| 看圖選字 | 顯示圖片，選出正確英文單字 | 正確率 ≥ 80% |
| 拼寫驗證 | 拖動字母拼出完整單字 | 正確率 ≥ 70% |
| 配對驗證 | 限時內完成英文-圖片配對 | 完成率 100%，翻牌次數 ≤ 最佳次數 ×1.5 |

### 學習進度追蹤

每個單字有獨立的熟練度指標：

| 等級 | 名稱 | 條件 |
|------|------|------|
| ⬜ | 未學習 | 尚未接觸此單字 |
| 🟨 | 認識中 | 在遊戲中正確辨認 1-2 次 |
| 🟩 | 熟悉 | 在不同遊戲中正確辨認 3-5 次 |
| ⭐ | 精熟 | 連續 3 次不同時段皆正確，且反應時間 ≤ 3 秒 |

### 複習機制

- **間隔重複**：根據艾賓浩斯遺忘曲線，在 1 天、3 天、7 天、14 天後自動安排複習
- **弱點加強**：錯誤率高的單字會在後續遊戲中提高出現頻率
- **每週小測驗**：每週自動生成一份包含本週所學單字的小測驗
