## 🎮 遊戲設計

### 遊戲：交通大冒險

**遊戲類型：** 🏃 RUNNER 跑酷收集 — 角色跑步收集正確單字

**遊戲說明：**

可愛角色在場景中自動向前跑，路上會出現浮動的單字氣泡。畫面上方顯示目標圖片，小朋友需要左右滑動角色去碰觸正確的英文單字，躲避錯誤單字。

**遊戲流程：**

1. 角色自動在場景中向前奔跑
2. 畫面上方顯示目標：一張圖片 + 中文提示
3. 路上出現漂浮的英文單字氣泡
4. 小朋友左右滑動螢幕控制角色移動方向
5. 碰到正確單字：加分 + 金幣特效
6. 碰到錯誤單字：扣除一顆愛心
7. 收集足夠正確單字即可過關

**難度分級：**

| 難度 | 參數設定 |
|------|----------|
| ⭐ Easy | 跑速慢，單字間距大，同時只出現 2 個選項，3 條跑道 |
| ⭐⭐ Medium | 跑速中等，單字間距縮小，同時出現 3 個選項，需要跳躍 |
| ⭐⭐⭐ Hard | 跑速快，同時出現 4 個選項，加入障礙物，限時模式 |

**Flame 技術實作要點：**

- 使用 `ParallaxComponent` 建立多層捲動背景
- 使用 `SpriteAnimationComponent` 製作角色跑步動畫
- 使用 `MoveEffect` 控制單字氣泡的浮動軌跡
- 使用 `HasCollisionDetection` + `RectangleHitbox` 偵測碰撞
- 透過 `JoystickComponent` 或手勢控制角色左右移動
- 使用 `SpawnComponent` 定時生成新的單字氣泡

**素材需求：**

- 角色素材：跑步動畫 Spritesheet（8 幀，512x512px）
- 背景素材：多層 Parallax 背景（配合交通與旅行主題）
- 氣泡素材：35 組單字氣泡圖片
- 音效素材：收集正確音效、碰到錯誤音效、跳躍音效、背景音樂
- 障礙物素材：各種障礙物圖片（中/高難度用）

### 遊戲：交通泡泡消

**遊戲類型：** 💥 POP 泡泡消除 — 戳破包含正確單字的泡泡

**遊戲說明：**

畫面中飄浮著許多彩色泡泡，每個泡泡上寫著一個英文單字。播放語音後，小朋友要快速點擊（戳破）包含正確單字的泡泡。泡泡會慢慢飄走，要在消失前戳破！

**遊戲流程：**

1. 播放目標單字的語音和顯示圖片
2. 多個彩色泡泡從畫面四周飄入
3. 每個泡泡上顯示不同的英文單字
4. 小朋友快速點擊正確的泡泡將其戳破
5. 戳破正確泡泡：爆炸星星特效 + 得分
6. 戳破錯誤泡泡：泡泡變灰 + 扣分提示
7. 限時內戳破越多正確泡泡，分數越高

**難度分級：**

| 難度 | 參數設定 |
|------|----------|
| ⭐ Easy | 泡泡移動慢，體積大，同時 3 個泡泡，目標泡泡顏色不同 |
| ⭐⭐ Medium | 泡泡移動速度中等，體積縮小，同時 5 個泡泡 |
| ⭐⭐⭐ Hard | 泡泡移動快，體積小，同時 7 個泡泡，泡泡會重疊 |

**Flame 技術實作要點：**

- 使用 `CircleComponent` 繪製泡泡形狀
- 使用 `RemoveEffect` + `ScaleEffect` 實現戳破動畫
- 使用 `MoveEffect` 讓泡泡隨機飄動
- 使用 `TextComponent` 在泡泡上顯示英文單字
- 使用 `OpacityEffect` 讓泡泡逐漸消失
- 透過 `TapCallbacks` 偵測點擊泡泡事件

**素材需求：**

- 泡泡素材：多種顏色泡泡圖片（PNG, 帶透明度）
- 文字素材：35 個單字的文字渲染
- 音效素材：泡泡破裂音效（pop sound）、背景泡泡音
- 粒子特效：泡泡爆破星星粒子、碎片動畫
- 圖片素材：35 組目標提示圖片

## 📊 學習目標驗證

### 驗證方式

| 驗證方法 | 說明 | 通過標準 |
|----------|------|----------|
| 聽音辨字 | 播放英文發音，選出正確圖片 | 正確率 ≥ 80% |
| 看圖選字 | 顯示圖片，選出正確英文單字 | 正確率 ≥ 80% |
| 拼寫驗證 | 拖動字母拼出完整單字 | 正確率 ≥ 70% |
| 配對驗證 | 限時內完成英文-圖片配對 | 完成率 100%，翻牌次數 ≤ 最佳次數 ×1.5 |

### 學習進度追蹤

每個單字有獨立的熟練度指標：

| 等級 | 名稱 | 條件 |
|------|------|------|
| ⬜ | 未學習 | 尚未接觸此單字 |
| 🟨 | 認識中 | 在遊戲中正確辨認 1-2 次 |
| 🟩 | 熟悉 | 在不同遊戲中正確辨認 3-5 次 |
| ⭐ | 精熟 | 連續 3 次不同時段皆正確，且反應時間 ≤ 3 秒 |

### 複習機制

- **間隔重複**：根據艾賓浩斯遺忘曲線，在 1 天、3 天、7 天、14 天後自動安排複習
- **弱點加強**：錯誤率高的單字會在後續遊戲中提高出現頻率
- **每週小測驗**：每週自動生成一份包含本週所學單字的小測驗
