## 🎮 遊戲設計

### 遊戲：句子建造師

**遊戲類型：** 🏗️ BUILD 拼圖組裝 — 拼出正確字母順序

**遊戲說明：**

畫面顯示一個目標單字的圖片和發音，下方散落著打亂順序的字母方塊。小朋友需要把字母方塊拖到正確的位置，拼出完整的單字。拼對後建築物會蓋起一層！

**遊戲流程：**

1. 畫面上方顯示目標圖片並播放單字發音
2. 下方顯示打亂順序的字母方塊
3. 小朋友拖動字母方塊到正確位置
4. 字母靠近正確位置時會自動吸附（Snap 效果）
5. 放錯位置的字母會彈回原處
6. 全部字母歸位後，建築物蓋起一層 + 慶祝動畫
7. 完成所有單字後展示完整建築物

**難度分級：**

| 難度 | 參數設定 |
|------|----------|
| ⭐ Easy | 3-4 個字母的單字，字母位置有淺色提示，無時間限制 |
| ⭐⭐ Medium | 5-6 個字母的單字，部分字母有位置提示，限時 30 秒 |
| ⭐⭐⭐ Hard | 7+ 個字母的單字，無位置提示，限時 20 秒 |

**Flame 技術實作要點：**

- 使用 `DragCallbacks` 實現字母方塊的拖動
- 自訂 `SnapEffect` 實現字母靠近正確位置時的自動吸附
- 使用 `TextComponent` 顯示字母和目標單字
- 使用 `SpriteComponent` 載入建築物各層圖片
- 使用 `MoveEffect` + `ScaleEffect` 實現建築物堆疊動畫
- 使用 `HasCollisionDetection` 偵測字母放置位置是否正確

**素材需求：**

- 字母素材：A-Z 字母方塊圖片（可愛積木風格）
- 建築素材：多層建築物圖片（配合連接詞與副詞主題）
- 圖片素材：42 組目標單字圖片
- 音效素材：拖動音效、吸附音效、建築完成音效
- 動畫素材：字母吸附動畫、建築堆疊動畫、完成慶祝煙火

### 遊戲：連接詞快手

**遊戲類型：** 🎯 TAP 點擊選擇 — 聽音選圖/看圖選字

**遊戲說明：**

畫面顯示多張圖片，播放英文單字語音，小朋友點擊對應的圖片。答對後圖片會發光並顯示獎勵動畫，答錯則輕微震動提示再試一次。隨著關卡進階，選項數量從 2 個增加到 4 個。

**遊戲流程：**

1. 系統播放英文單字語音
2. 畫面顯示 2~4 張圖片選項
3. 小朋友點擊認為正確的圖片
4. 正確：圖片放大 + 星星特效 + 播放歡呼音效
5. 錯誤：圖片輕微震動 + 顯示正確答案 + 鼓勵提示
6. 連續答對可獲得連擊加分
7. 完成一輪後顯示成績單與獎勵

**難度分級：**

| 難度 | 參數設定 |
|------|----------|
| ⭐ Easy | 顯示 2 個選項，圖片差異大，語音播放 2 次，無時間限制 |
| ⭐⭐ Medium | 顯示 3 個選項，圖片相似度提高，語音播放 1 次，每題 10 秒 |
| ⭐⭐⭐ Hard | 顯示 4 個選項，加入相似發音的干擾選項，每題 6 秒 |

**Flame 技術實作要點：**

- 使用 `SpriteComponent` 載入圖片選項，設定 `size` 適配螢幕
- 實作 `TapCallbacks` mixin 處理點擊事件
- 使用 `ScaleEffect` 實現點擊時的放大縮小動畫
- 使用 `ColorEffect` 實現答對/答錯的顏色回饋
- 透過 `AudioPool` 預載音效，搭配 `FlameAudio` 播放語音
- 使用 `TimerComponent` 實作倒數計時（中/高難度）

**素材需求：**

- 圖片素材：42 組單字對應圖片（PNG, 256x256px）
- 音效素材：正確音效、錯誤音效、點擊音效
- 語音素材：所有單字的英文發音（MP3）
- 動畫素材：星星粒子特效、答對慶祝動畫
- UI 素材：計時器圖示、分數面板、愛心圖示

## 🏆 獎勵設計

### 即時獎勵
- ⭐ **星星**：每答對一題獲得 1-3 顆星星（依難度）
- 🎖️ **連擊獎勵**：連續答對 3 題以上，獲得額外金幣
- 💎 **完美獎勵**：單局全部答對，獲得鑽石 ×1

### 收集獎勵
- 🖼️ **主題收集**：收集連接積木，建造超級長句子
- 每學會一個單字，解鎖一個收集品
- 收集品可在「我的收藏」頁面查看

### 成就獎勵

- 🥈 **銀牌**：完成 Medium 難度全部關卡
- 🥇 **金牌**：完成 Hard 難度全部關卡
- 👑 **王冠**：所有單字達到「精熟」等級

## 📊 學習目標驗證

### 驗證方式

| 驗證方法 | 說明 | 通過標準 |
|----------|------|----------|
| 聽音辨字 | 播放英文發音，選出正確圖片 | 正確率 ≥ 80% |
| 看圖選字 | 顯示圖片，選出正確英文單字 | 正確率 ≥ 80% |
| 拼寫驗證 | 拖動字母拼出完整單字 | 正確率 ≥ 70% |
| 配對驗證 | 限時內完成英文-圖片配對 | 完成率 100%，翻牌次數 ≤ 最佳次數 ×1.5 |

### 學習進度追蹤

每個單字有獨立的熟練度指標：

| 等級 | 名稱 | 條件 |
|------|------|------|
| ⬜ | 未學習 | 尚未接觸此單字 |
| 🟨 | 認識中 | 在遊戲中正確辨認 1-2 次 |
| 🟩 | 熟悉 | 在不同遊戲中正確辨認 3-5 次 |
| ⭐ | 精熟 | 連續 3 次不同時段皆正確，且反應時間 ≤ 3 秒 |

### 複習機制

- **間隔重複**：根據艾賓浩斯遺忘曲線，在 1 天、3 天、7 天、14 天後自動安排複習
- **弱點加強**：錯誤率高的單字會在後續遊戲中提高出現頻率
- **每週小測驗**：每週自動生成一份包含本週所學單字的小測驗
