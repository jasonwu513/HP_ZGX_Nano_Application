## 🎮 遊戲設計

### 遊戲：美食接接樂

**遊戲類型：** 🎪 CATCH 接住落下 — 用籃子接住正確單字

**遊戲說明：**

各種物品從畫面上方落下，每個物品上標示英文單字。小朋友左右移動底部的籃子，接住老師指定的正確單字物品，避開錯誤的物品。速度會逐漸加快！

**遊戲流程：**

1. 畫面上方開始掉落帶有英文單字的物品
2. 畫面頂部顯示目標：要接住的單字（圖片+中文提示）
3. 小朋友左右滑動底部籃子移動
4. 接住正確物品：籃子發光 + 加分
5. 接住錯誤物品：籃子震動 + 扣一顆心
6. 漏接正確物品不扣分但不加分
7. 每接住 5 個正確單字，速度略微提升

**難度分級：**

| 難度 | 參數設定 |
|------|----------|
| ⭐ Easy | 物品下落速度慢，同時 1 個物品，籃子寬度大 |
| ⭐⭐ Medium | 下落速度中等，同時 2 個物品，籃子寬度中等 |
| ⭐⭐⭐ Hard | 下落速度快，同時 3 個物品，籃子寬度小，物品會左右飄動 |

**Flame 技術實作要點：**

- 使用 `MoveEffect` 控制物品從上方落下的動畫
- 使用 `CollisionCallbacks` + `RectangleHitbox` 偵測籃子接住物品
- 使用 `DragCallbacks` 或觸控事件控制籃子左右移動
- 使用 `SpawnComponent` 定時從隨機位置生成下落物品
- 使用 `SpriteComponent` 載入物品和籃子的圖片素材
- 使用 `RemoveEffect` 移除已接住或掉落的物品

**素材需求：**

- 物品素材：55 組帶有單字的下落物品圖片
- 籃子素材：可愛籃子/容器圖片（配合主題設計）
- 音效素材：接住音效、漏接音效、背景音樂
- 動畫素材：接住成功閃光效果、物品落地碎裂效果
- UI 素材：分數面板、愛心生命值、速度指示器

### 遊戲：美食跑跑跑

**遊戲類型：** 🏃 RUNNER 跑酷收集 — 角色跑步收集正確單字

**遊戲說明：**

可愛角色在場景中自動向前跑，路上會出現浮動的單字氣泡。畫面上方顯示目標圖片，小朋友需要左右滑動角色去碰觸正確的英文單字，躲避錯誤單字。

**遊戲流程：**

1. 角色自動在場景中向前奔跑
2. 畫面上方顯示目標：一張圖片 + 中文提示
3. 路上出現漂浮的英文單字氣泡
4. 小朋友左右滑動螢幕控制角色移動方向
5. 碰到正確單字：加分 + 金幣特效
6. 碰到錯誤單字：扣除一顆愛心
7. 收集足夠正確單字即可過關

**難度分級：**

| 難度 | 參數設定 |
|------|----------|
| ⭐ Easy | 跑速慢，單字間距大，同時只出現 2 個選項，3 條跑道 |
| ⭐⭐ Medium | 跑速中等，單字間距縮小，同時出現 3 個選項，需要跳躍 |
| ⭐⭐⭐ Hard | 跑速快，同時出現 4 個選項，加入障礙物，限時模式 |

**Flame 技術實作要點：**

- 使用 `ParallaxComponent` 建立多層捲動背景
- 使用 `SpriteAnimationComponent` 製作角色跑步動畫
- 使用 `MoveEffect` 控制單字氣泡的浮動軌跡
- 使用 `HasCollisionDetection` + `RectangleHitbox` 偵測碰撞
- 透過 `JoystickComponent` 或手勢控制角色左右移動
- 使用 `SpawnComponent` 定時生成新的單字氣泡

**素材需求：**

- 角色素材：跑步動畫 Spritesheet（8 幀，512x512px）
- 背景素材：多層 Parallax 背景（配合食物與飲料主題）
- 氣泡素材：55 組單字氣泡圖片
- 音效素材：收集正確音效、碰到錯誤音效、跳躍音效、背景音樂
- 障礙物素材：各種障礙物圖片（中/高難度用）

## 🏆 獎勵設計

### 即時獎勵
- ⭐ **星星**：每答對一題獲得 1-3 顆星星（依難度）
- 🎖️ **連擊獎勵**：連續答對 3 題以上，獲得額外金幣
- 💎 **完美獎勵**：單局全部答對，獲得鑽石 ×1

### 收集獎勵
- 🖼️ **主題收集**：收集食材卡片，完成美味食譜書
- 每學會一個單字，解鎖一個收集品
- 收集品可在「我的收藏」頁面查看

### 成就獎勵
- 🥉 **銅牌**：完成 Easy 難度全部關卡
- 🥈 **銀牌**：完成 Medium 難度全部關卡
- 🥇 **金牌**：完成 Hard 難度全部關卡
- 👑 **王冠**：所有單字達到「精熟」等級

## 📊 學習目標驗證

### 驗證方式

| 驗證方法 | 說明 | 通過標準 |
|----------|------|----------|
| 聽音辨字 | 播放英文發音，選出正確圖片 | 正確率 ≥ 80% |
| 看圖選字 | 顯示圖片，選出正確英文單字 | 正確率 ≥ 80% |
| 拼寫驗證 | 拖動字母拼出完整單字 | 正確率 ≥ 70% |
| 配對驗證 | 限時內完成英文-圖片配對 | 完成率 100%，翻牌次數 ≤ 最佳次數 ×1.5 |

### 學習進度追蹤

每個單字有獨立的熟練度指標：

| 等級 | 名稱 | 條件 |
|------|------|------|
| ⬜ | 未學習 | 尚未接觸此單字 |
| 🟨 | 認識中 | 在遊戲中正確辨認 1-2 次 |
| 🟩 | 熟悉 | 在不同遊戲中正確辨認 3-5 次 |
| ⭐ | 精熟 | 連續 3 次不同時段皆正確，且反應時間 ≤ 3 秒 |

### 複習機制

- **間隔重複**：根據艾賓浩斯遺忘曲線，在 1 天、3 天、7 天、14 天後自動安排複習
- **弱點加強**：錯誤率高的單字會在後續遊戲中提高出現頻率
- **每週小測驗**：每週自動生成一份包含本週所學單字的小測驗
