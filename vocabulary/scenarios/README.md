# 小一英語情境學習教案總目錄

> 適合對象：小一學生 (6-7 歲) ｜ 總單字數：1,200 個 ｜ 教案數：32 份

本系列教案以**遊戲導向**設計，將 1,200 個基礎英語單字依主題分類，每份教案包含：

- 單字列表（含中文翻譯與簡單說明）
- 故事開場（情境引導）
- 3-4 個互動遊戲
- 唱唱跳跳（韻律歌謠）
- 三關闖關挑戰（聽力 → 配對 → 句子）
- 回家小任務（親子互動）

## 遊戲類型速查表

| 遊戲類型 | 說明 | 適用教案 |
|----------|------|----------|
| 圖卡配對 | 翻牌配對英文與圖片 | 動物、食物、衣服、交通 |
| 大風吹 | 「大風吹，吹有 cat 的人」 | 動物、顏色、衣服 |
| Simon Says | 老師說指令，學生做動作 | 身體、動作動詞、方向 |
| 超市購物 | 角色扮演買東西 | 食物、購物、數字 |
| 猜猜我是誰 | 描述特徵讓人猜 | 職業、動物、家人 |
| 比手畫腳 | 表演單字讓人猜 | 動詞、感覺、運動 |
| 尋寶遊戲 | 在教室找隱藏的單字卡 | 居家、學校、介系詞 |
| 接力賽 | 分組搶答或接力拼字 | 任何分類 |
| 歌曲律動 | 配合音樂唱跳 | 身體、數字、天氣 |
| 故事接龍 | 用單字接力編故事 | 社交用語、形容詞 |
| Bingo 賓果 | 聽英文劃掉圖片 | 任何分類 |
| 角色扮演 | 扮演特定場景 | 職業、購物、城鎮 |

## 建議使用順序

1. **入門**：先從動物 (04)、食物 (03)、顏色 (05) 開始，這些最具體、最有趣
2. **身體互動**：身體 (02)、衣服 (08)、感覺 (13)，搭配 TPR 教學法
3. **生活場景**：居家 (09)、學校 (10)、交通 (12)、城鎮 (15)、職業 (16)
4. **數字時間**：數字 (06)、時間 (07)、天氣 (11)
5. **動詞形容詞**：動詞系列 (14-1~4)、形容詞系列 (20-1~4)
6. **進階補充**：其餘教案視學生程度選用
