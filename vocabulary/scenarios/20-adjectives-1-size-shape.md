# 形容詞：大小形狀 Adjectives: Size & Shape — 遊戲學習情境

> 🎯 目標單字：21 個（#771-#791）｜ 適合對象：小一學生 (6-7歲)

---

## 單字列表

### 📏 大小與長短 Size & Length（5 個）

| 英文 | 中文 | 簡單說明 |
|------|------|----------|
| big | 大的 | large in size |
| small | 小的 | little in size |
| tall | 高的 | high from bottom to top |
| short | 矮的 | not tall; not long |
| long | 長的 | having a great length |

| 英文 | 中文 | 簡單說明 |
|------|------|----------|
| wide | 寬的 | having a large distance from side to side |
| narrow | 窄的 | having a small distance from side to side |
| thick | 厚的 | having a large distance between two sides |
| thin | 薄的 | having a small distance between two sides |

| 英文 | 中文 | 簡單說明 |
|------|------|----------|
| heavy | 重的 | weighing a lot; hard to lift |

## 🎬 故事開場：反義詞王國大冒險

老師拿出一個大紙箱和一個小紙盒，用誇張的語氣說：

> 「小朋友們，今天老師要帶你們去一個很特別的地方——**反義詞王國**！」
>
> 「在反義詞王國裡，所有東西都是成對出現的。你看！這是一個 **big** box（大箱子），旁邊還有一個 **small** box（小盒子）。Big 和 small 是好朋友，也是相反的！」
> *(老師把大箱子舉高，再拿起小盒子，讓孩子看到對比)*
>
> 「反義詞王國的國王是一個很 **tall**（高）的巨人，他的好朋友是一個很 **short**（矮）的小精靈。巨人有一條 **long**（長）的圍巾，小精靈的圍巾很 **short**（短）。」
> *(老師用手比出高和矮的樣子，再拿一條長絲巾和一條短布條)*
>
> 「他們住的路也不一樣——巨人走的是 **wide**（寬）的大馬路，小精靈走的是 **narrow**（窄）的小巷子。」
> *(老師用兩隻手臂張開表示「寬」，再縮到很近表示「窄」)*
>
> 「今天巨人和小精靈要去一個神祕的山洞冒險！山洞的門是 **thick**（厚）的大石頭做的，可是旁邊有一面 **thin**（薄）的紙牆。巨人搬了一塊 **heavy**（重）的石頭，小精靈撿了一片 **light**（輕）的羽毛。」
> *(老師假裝搬很重的東西，表情用力；再假裝拿一片羽毛，輕飄飄的)*
>
> 「山洞裡面，巨人摸到牆壁，有的地方 **smooth**（光滑的），有的地方 **rough**（粗糙的）。地上有 **hard**（硬）的石頭，也有 **soft**（軟）的青苔。」
> *(老師做出摸東西的動作，一會兒皺眉說 rough，一會兒微笑說 smooth)*
>
> 「山洞裡有一條河，有的地方 **deep**（深），有的地方 **shallow**（淺）。巨人發現河邊有一塊 **flat**（平）的石頭和一把 **sharp**（鋒利）的劍！小精靈看到旁邊有一個寶箱——打開來！裡面是 **full**（滿）的金幣！可是另一個寶箱打開來卻是 **empty**（空）的，什麼都沒有！」
> *(老師做出「哇好多金幣！」的驚喜表情，再做出空空如也的失望表情)*
>
> 「好！小朋友們，今天我們就要跟著巨人和小精靈，一起認識這些反義詞！準備好了嗎？」

老師在黑板上畫出巨人和小精靈的簡圖，每學會一組反義詞，就在他們中間寫上那對單字，最後黑板上會出現完整的「反義詞地圖」。

## 🎮 遊戲活動

### 遊戲一：反義詞大對決

**目標單字：** big/small, tall/short, long/short, wide/narrow, thick/thin, heavy/light, hard/soft, smooth/rough, deep/shallow, full/empty, sharp/flat

**準備材料：**
- 21 張單字卡（正面英文和中文，背面統一圖案）
- 另備 10 張「反義詞配對卡」（每張上面印有一對反義詞的插圖）

**遊戲規則：**
1. 把 21 張單字卡洗牌後，面朝下排成 3 行 7 列。
2. 小朋友輪流翻開兩張卡片，如果翻到的兩張剛好是一對反義詞（例如 big 和 small），就可以收走這一對，並大聲唸出來：**"Big and small! They are opposites!"**
3. 如果翻到的兩張不是一對反義詞，就翻回去蓋住，換下一位。
4. sharp 和 flat 雖然不是嚴格的反義詞，但在本遊戲中設定為一組（尖的 vs. 平的），方便配對。long 和 short 也可以配成一對。
5. 最後收集最多對的小朋友獲勝！

**範例對話：**
- 翻到 big 和 small → 老師引導：**"Yes! Big is the opposite of small. A big dog and a small cat!"**
- 翻到 heavy 和 light → 老師引導：**"Heavy and light! This book is heavy. This feather is light."**
- 翻到 full 和 rough → 不配對，翻回去。

**教學重點：**
- 每配對成功一次，全班一起說一個反義詞短句：**"The elephant is big. The mouse is small."**
- 讓孩子反覆看到成對的反義詞，強化記憶。

---

### 遊戲二：觸摸猜猜看

**目標單字：** hard, soft, smooth, rough, heavy, light, thick, thin, sharp, flat

**準備材料：**
- 一個不透光的大布袋（「神祕觸摸袋」）
- 10 種實物放進袋子裡：
  - **hard** → 一塊石頭或積木
  - **soft** → 一團棉花或絨毛娃娃
  - **smooth** → 一顆玻璃彈珠或光滑塑膠球
  - **rough** → 一張砂紙或粗麻繩
  - **heavy** → 一本厚字典或裝水的水壺
  - **light** → 一片羽毛或一張紙
  - **thick** → 一本厚書
  - **thin** → 一張薄紙或塑膠片
  - **sharp** → 一支削好的鉛筆（注意安全）
  - **flat** → 一把尺或一片平整的硬紙板

**遊戲規則：**
1. 小朋友閉上眼睛（或用布矇住眼睛），把手伸進神祕觸摸袋裡摸一樣東西。
2. 老師問：**"How does it feel? Is it hard or soft? Is it smooth or rough? Is it heavy or light?"**
3. 小朋友用英文回答自己摸到的感覺：**"It feels hard!"** 或 **"It is smooth!"**
4. 回答完後，猜猜摸到的是什麼東西，再把東西拿出來看看是否猜對。
5. 回答和猜測都正確的小朋友獲得兩顆星星，只答對觸感的獲得一顆。

**範例對話：**
- 小朋友：**"It feels soft and light."**
- 老師：**"Soft and light! What do you think it is?"**
- 小朋友：**"A feather?"**
- 老師：**"Let's see... Yes! It's a feather! A feather is soft and light. Great job!"**

**教學重點：**
- 讓孩子用手直接感受形容詞的意義，觸覺學習印象最深刻。
- 鼓勵孩子組合兩個形容詞描述：**"It is hard and smooth."**（例如摸到彈珠）

### 遊戲三：教室裡的形容詞尋寶

**目標單字：** big, small, tall, short, long, wide, narrow, heavy, light, hard, soft, full, empty, thick, thin

**準備材料：**
- 每組一張「尋寶清單」學習單
- 一支筆
- 計時器（限時 5 分鐘）

**尋寶清單範例：**

| 找到一樣東西是... | 我找到的東西 | 英文句子 |
|-------------------|-------------|----------|
| **big**（大的） | 例：黑板 | The blackboard is big. |
| **small**（小的） | | |
| **tall**（高的） | | |
| **short**（矮的） | | |
| **long**（長的） | | |
| **heavy**（重的） | | |
| **light**（輕的） | | |
| **hard**（硬的） | | |
| **soft**（軟的） | | |
| **full**（滿的） | | |
| **empty**（空的） | | |
| **thick**（厚的） | | |
| **thin**（薄的） | | |
| **wide**（寬的） | | |
| **narrow**（窄的） | | |

**遊戲規則：**
1. 全班分成 4-5 組，每組 4-5 人。
2. 每組拿到一張尋寶清單。限時 5 分鐘，在教室裡找到符合每個形容詞的物品。
3. 找到後，在學習單上寫下物品名稱，並試著用英文造一個句子。
4. 時間到後，每組上台發表找到的東西，用英文唸出句子：
   - **"The desk is hard."**
   - **"The eraser is soft."**
   - **"The pencil case is full."**
   - **"The box is empty."**
5. 找到最多樣、句子說得最好的那一組獲勝。

**教學重點：**
- 讓孩子走動觀察教室，把抽象的形容詞和生活中的真實物品連結。
- 可能有不同答案（例如「桌子」可以是 big，也可以是 hard），鼓勵多元思考。
- 每組發表時，全班跟著複誦一遍句子。

### 遊戲四：形容詞 Bingo

**目標單字：** 全部 21 個單字

**準備材料：**
- 每位小朋友一張 4x4 的 Bingo 格子紙（共 16 格）
- 21 個單字卡片（老師抽用）
- 彩色圓點貼紙或蠟筆（用來標記）

**遊戲規則：**
1. 老師把 21 個單字寫在黑板上，小朋友從中挑 16 個寫在自己的 Bingo 紙上（一格一個字，可以自由排列）。
2. 老師抽一張單字卡，不直接唸出單字，而是用描述或反義詞提示：
   - 老師說：**"This is the opposite of big."** → 答案是 **small**
   - 老師說：**"A rock feels like this. It is not soft."** → 答案是 **hard**
   - 老師說：**"When your cup has no water, it is..."** → 答案是 **empty**
   - 老師說：**"A knife is this. Be careful!"** → 答案是 **sharp**
   - 老師說：**"The swimming pool is very... You cannot touch the bottom."** → 答案是 **deep**
3. 小朋友聽懂後，在自己紙上找到那個字並貼上圓點或打圈。
4. 第一個連成一條線（橫、直、斜都可以）的小朋友喊 **"Bingo!"** 並把連線的四個字唸出來。

**範例對話：**
- 老師：**"This word means something has no space inside. Your backpack after you put all the books in — it is very..."**
- 小朋友：**"Full!"**（在自己的 Bingo 紙上標記 full）

**教學重點：**
- 用描述和反義詞提示代替直接唸單字，訓練孩子的聽力理解。
- Bingo 完成後，可以再玩一輪，讓更多孩子有機會贏。
- 最後全班一起把 21 個單字大聲唸一遍做總複習。

## 🎤 唱唱跳跳：Opposites Song（反義詞之歌）

用簡單重複的旋律（類似 "Are You Sleeping / Frere Jacques" 的節奏），搭配身體動作：

### 歌詞與動作

> 🎵 **第一段：**
> **Big, big, big!** *(雙手張開畫大圈)*
> **Small, small, small!** *(雙手縮小比一個小圈)*
> **Big and small, big and small,** *(交替比大圈和小圈)*
> **Opposites! Opposites!** *(雙手交叉再打開)*
>
> 🎵 **第二段：**
> **Tall, tall, tall!** *(踮腳尖把手舉到最高)*
> **Short, short, short!** *(蹲下來縮小)*
> **Tall and short, tall and short,**
> **Opposites! Opposites!**
>
> 🎵 **第三段：**
> **Heavy, heavy, heavy!** *(假裝搬很重的東西，彎腰用力)*
> **Light, light, light!** *(假裝拿羽毛，輕輕飄)*
> **Heavy and light, heavy and light,**
> **Opposites! Opposites!**
>
> 🎵 **第四段：**
> **Hard, hard, hard!** *(用拳頭敲桌子)*
> **Soft, soft, soft!** *(用手輕輕摸臉頰)*
> **Hard and soft, hard and soft,**
> **Opposites! Opposites!**
>
> 🎵 **第五段：**
> **Smooth, smooth, smooth!** *(用手掌在桌面上滑動)*
> **Rough, rough, rough!** *(手指抓抓搓搓)*
> **Smooth and rough, smooth and rough,**
> **Opposites! Opposites!**
>
> 🎵 **第六段：**
> **Full, full, full!** *(雙手在肚子前畫一個大圓，裝得滿滿的)*
> **Empty, empty, empty!** *(雙手攤開，什麼都沒有)*
> **Full and empty, full and empty,**
> **Opposites! Opposites!**
>
> 🎵 **最後一段（大結尾）：**
> **Long and short! Wide and narrow!** *(比長再比短，張開再縮窄)*
> **Thick and thin! Deep and shallow!** *(比厚再比薄，手往下壓再往上提)*
> **Sharp and flat! We know them all!** *(比尖角再比平面)*
> **Opposites are so much fun!** *(全班歡呼拍手)*

### 動作總整理

| 反義詞組 | 動作 |
|----------|------|
| **big / small** | 雙手畫大圈 → 縮成小圈 |
| **tall / short** | 踮腳舉手 → 蹲下縮小 |
| **heavy / light** | 假裝搬重物 → 假裝拿羽毛 |
| **hard / soft** | 拳頭敲桌子 → 輕摸臉頰 |
| **smooth / rough** | 手掌滑桌面 → 手指搓搓 |
| **full / empty** | 肚子前畫圓 → 雙手攤開 |
| **long / short** | 雙手拉開 → 雙手靠近 |
| **wide / narrow** | 雙臂張開 → 雙臂靠近 |
| **thick / thin** | 雙手比出厚度 → 雙手靠近比薄 |
| **deep / shallow** | 手往下壓到地板 → 手輕輕放低 |
| **sharp / flat** | 手指比出尖角 → 手掌壓平 |

**教學小提醒：**
- 先一段一段慢慢教動作和歌詞，確保每個孩子都跟上。
- 熟練後可以加速唱，或讓男生唱第一個詞、女生唱反義詞。
- 每堂課開始唱一遍當暖身，兩三次之後孩子就能背起來了。

## 🏆 闖關挑戰

### 第一關：聽力挑戰 — 「相反耳朵」

**規則：**
1. 老師唸一個形容詞，小朋友要立刻大聲說出它的反義詞。
2. 共出 12 題，答對 8 題以上即過關。

**題目範例：**

| 題號 | 老師說 | 小朋友要回答 |
|------|--------|-------------|
| 1 | **"Big!"** | **"Small!"** |
| 2 | **"Tall!"** | **"Short!"** |
| 3 | **"Heavy!"** | **"Light!"** |
| 4 | **"Hard!"** | **"Soft!"** |
| 5 | **"Smooth!"** | **"Rough!"** |
| 6 | **"Full!"** | **"Empty!"** |
| 7 | **"Deep!"** | **"Shallow!"** |
| 8 | **"Wide!"** | **"Narrow!"** |
| 9 | **"Thick!"** | **"Thin!"** |
| 10 | **"Long!"** | **"Short!"** |
| 11 | **"Sharp!"** | **"Flat!"** |
| 12 | **"Small!"** | **"Big!"** |

**進階玩法：** 老師越唸越快，或改用情境描述：
- **"My bag is very... I can't carry it!"** → **"Heavy!"**
- **"The glass has no water. It is..."** → **"Empty!"**

**過關獎勵：** 獲得一顆「反義詞之星」貼紙。

---

### 第二關：配對挑戰 — 「反義詞連連看」

**規則：**
1. 黑板左邊貼 11 張單字卡，右邊貼對應的 11 張反義詞卡（打亂順序）。
2. 小朋友輪流上台連線，連線時要大聲唸出兩個詞：**"Big and small — they are opposites!"**
3. 全部連對即過關。

**配對內容（共 11 組）：**

| 左邊 | 右邊（反義詞） |
|------|----------------|
| big | small |
| tall | short |
| long | short |
| wide | narrow |
| thick | thin |
| heavy | light |
| hard | soft |
| smooth | rough |
| deep | shallow |
| full | empty |
| sharp | flat |

**注意：** short 同時是 tall 和 long 的反義詞，老師可以趁機說明：**"Short is special! It is the opposite of tall AND long!"**

**過關獎勵：** 獲得一顆「配對達人」貼紙。

---

### 第三關：句子挑戰 — 「我會用反義詞造句」

**規則：**
1. 老師給小朋友看一張圖片或實物，小朋友要用對應的形容詞造出完整的英文句子。
2. 黑板上提供句型模板：
   - **"The ___ is ___."**（這個⋯⋯是⋯⋯的。）
   - **"It is ___, not ___."**（它是⋯⋯，不是⋯⋯。）
   - **"This one is ___, but that one is ___."**（這個是⋯⋯，但那個是⋯⋯。）
3. 共 10 題，能說出 6 句以上即過關。

**題目範例：**

| 題號 | 老師展示的東西 | 參考答案 |
|------|---------------|----------|
| 1 | 一顆大球和一顆小球 | **"This ball is big, but that ball is small."** |
| 2 | 一杯裝滿水的杯子 | **"The cup is full."** |
| 3 | 一個空的鉛筆盒 | **"The pencil case is empty."** |
| 4 | 一塊石頭 | **"The rock is hard and rough."** |
| 5 | 一隻絨毛娃娃 | **"The teddy bear is soft and light."** |
| 6 | 一把長尺和一把短尺 | **"This ruler is long. That ruler is short."** |
| 7 | 一本很厚的書 | **"The book is thick and heavy."** |
| 8 | 一張紙 | **"The paper is thin and light."** |
| 9 | 一個寬寬的桌子 | **"The table is wide and flat."** |
| 10 | 一支削好的鉛筆 | **"The pencil is sharp. Be careful!"** |

**過關獎勵：** 獲得一顆「句子高手」貼紙。集滿三關貼紙可獲得「反義詞小達人」獎狀！

## 📝 回家小任務：我的反義詞小書

請小朋友回家完成以下任務，下次上課帶來跟大家分享：

### 任務一：反義詞圖畫書（畫 + 寫）

用一張 A4 紙對摺，做成一本小書。每一頁畫出一組反義詞的對比圖，旁邊寫上英文單字和一句短句。

範例：
- 第一頁：左邊畫一隻大象，寫 **"The elephant is big."**；右邊畫一隻螞蟻，寫 **"The ant is small."**
- 第二頁：左邊畫一杯裝滿果汁的杯子，寫 **"The cup is full."**；右邊畫一個空杯子，寫 **"The cup is empty."**
- 至少畫 **4 組**反義詞（8 頁），越多越好！

> 💡 **給家長的話：** 這週我們學了 21 個描述大小、形狀和物理特徵的英文形容詞，重點放在「反義詞配對」的概念。在日常生活中，您可以隨時帶孩子練習——看到大車就說 **"A big car!"**，看到小貓就說 **"A small cat!"**；喝水時問 **"Is your cup full or empty?"**；摸東西時問 **"Is it hard or soft? Smooth or rough?"** 透過五感體驗，孩子會更容易記住這些形容詞。反義詞是學英文的好工具，成對記憶比單獨記更有效率！
